# GuessWord

**GuessWord** — это современное веб-приложение для угадывания слов, реализующее одиночный и многопользовательский режимы. Проект сочетает интеллектуальную игровую логику на основе векторных представлений слов (embeddings), безопасную авторизацию, хранение истории игр и таблицу лидеров.

## 🕹 Основной функционал

- Регистрация и авторизация пользователей с использованием JWT.
- Одиночный режим игры с ранжированием введённых слов по степени близости к загаданному слову.
- Многопользовательский режим с комнатами и синхронизацией через SignalR.
- Хранение истории игр и формирование таблицы лидеров.
- Подбор слов и вычисление рангов с использованием embeddings и косинусной близости.
- Кэширование рангов для ускорения обработки запросов.

## 📦 Архитектура проекта

Проект разделён на три ключевых слоя:

1. **Клиентская часть (Blazor WebAssembly)**
   - Отображение интерфейса и маршрутизация между страницами.
   - Обработка действий пользователя и отправка запросов к серверу.
   - SignalR для многопользовательского взаимодействия.
   - Хранение JWT и состояния пользователя в `localStorage`.

2. **Серверная часть (ASP.NET Core Web API)**
   - Обработка HTTP-запросов и бизнес-логики.
   - Авторизация и регистрация пользователей.
   - Управление одиночными и многопользовательскими играми.
   - Расчёт рангов слов по embeddings и кэширование.
   - SignalR-хаб для синхронизации событий между игроками.

3. **База данных (PostgreSQL + pgvector)**
   - Таблицы: `Users`, `Rooms`, `Games`, `GamePlayers`, `GameAttempts`, `Words`.
   - Поддержка хранения векторных представлений слов.
   - Сохранение истории игр и статистики игроков.

4. **Scripts**
   - Скрипты для автоматизации задач: заполнение базы данных начальными данными, миграции, генерация словарей, утилиты администрирования.

## 🚀 Установка и запуск

### Локальный запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/elon1te9/GuessWord.git
cd GuessWord
```

2. Настройте `.env` файл:
```env
ConnectionStrings__DefaultConnection="Host=localhost;Port=5432;Database=guessword;Username=postgres;Password=yourpassword"
Jwt__Key="your_secret_key"
```

3. Создайте и обновите базу данных:
```bash
dotnet ef database update --project GuessWord.Api
```

4. Запустите сервер:
```bash
dotnet run --project GuessWord.Api
```

5. Запустите клиент:
```bash
dotnet run --project GuessWord.Client
```

6. Откройте в браузере:
```
http://localhost:5000
```

### Развёртывание через Docker

1. Поднимите все сервисы:
```bash
docker-compose -f docker-compose.server.yml up -d
```

2. Приложение доступно по адресу сервера (в данный момент http://80.249.150.207).

## 🧩 Игровая логика

- Загаданное слово выбирается случайно из `secret-words.txt`.
- Каждое слово имеет embedding в 800-мерном пространстве.
- Ранжирование попыток осуществляется по косинусной близости к секретному слову.
- Побеждает игрок, первым угадавший слово.
- В многопользовательской игре обновления происходят в реальном времени через SignalR.
